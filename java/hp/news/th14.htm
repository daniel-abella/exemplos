Como fazer um thread "dormir"

Os métodos sleep(long) e sleep(long,int) dizem para um thread que é hora de ir para cama e permanecer lá, sem fazer absolutamente nada, o tempo fornecido como argumento. No primeiro caso, o tempo é fornecido em milissegundos. No segundo há um controle mais fino, e o valor em milissegundos é adicionado àquele em nanosegundos. Os dois trechos de código abaixo são equivalentes e ilustram o emprego destes métodos.

try {
    Thread.sleep(5*1000); // Dorme por 5 segundos
} catch (InterrupedException ex) {
    // Trate a exceção aqui!
}

Alternativamente, o código acima pode ser escrito como abaixo:

try {
    Thread.sleep(5*1000,0); // Dorme 5 segundos + 0 nanossegundos
} catch (InterrupedException ex) {
    // Trate a exceção aqui!
}
