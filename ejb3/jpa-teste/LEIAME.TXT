-----------------------------------
Código de teste para camada de negócio
-----------------------------------

OBSERVAÇÕES:

- Exige emprego de JMock para testar código que faz uso de 
  repositories. Lembrete: a implementação de repositórios faz
  parte da persistência e não da camada de domínio.
- Projeto de teste (<projeto>-teste)
- Projeto testado (<projeto>)
- Código de teste é fornecido no diretório 'test'
- Se a.b.X é a classe testada, então a.b.XTest é a classe de teste
- a.b.XTest está no diretório 'test'
- a.b.X está no diretório 'src' no projeto testado
