Criação de um thread via implementação de Runnable

O primeiro passo da criação de um thread é estender a classe java.lang.Thread ou implementar a interface java.lang.Runnable. O código abaixo faz uso da segunda opção.

public class MinhaTarefa implements Runnable {
    public void run() {
        /* Operação concorrente aqui! */
    }
}

Quando uma instância de MinhaTarefa é criada o método run() não é executado concorrentemente. Para que o método run() possa ser executado de forma concorrente é necessário criar uma instância da classe Thread e fornecer ao construtor uma instância da classe MinhaTarefa, conforme abaixo.

Thread meuThread = new Thread(new MinhaTarefa());

Isto ainda não é suficiente. Até então, uma instância de Thread foi criada e a tarefa a ser executada concorrentemente foi identificada. Contudo, nenhum thread ainda foi criado, apenas objetos em memória foram criados. Para que um thread seja criado é necessário enviar a mensagem start() para o objeto meuThread, conforme abaixo.

meuThread.start(); // Novo thread é criado!

Após a execução da sentença acima, um novo thread estará disponível para execução. A função do novo thread será executar o método run(), fornecido pelo objeto que implementa Runnable.
