Programa seqüencial

Um programa é dito seqüencial quando uma única tarefa deve ser executada de forma não concorrente. Por exemplo, o método void tarefa(String) do programa abaixo é executado duas vezes. Apenas ao término da primeira chamada a segunda será iniciada.

public class Sequencial {

    public void tarefa(String prefixo) {
        for (int i = 0; i < 1000; i++) {
            System.out.println(prefixo + " " + i);
        }
    }
                             
    public static void main(String[] args) {
        Sequencial sq = new Sequencial();
        sq.tarefa("1");
        sq.tarefa("2");
    }
}
