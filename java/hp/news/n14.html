NaN ???

Em Java, os tipos ponto flutuante, uma espécia de "aproximação dos Reais da matemática", são os tipos float e double. O float é de precisão simples (32 bits) e o double de precisão dupla (64 bits). Todos os valores possíveis associados a tais tipos, assim como as operações correspondentes estão devidamente documentas em IEEE Standard for Binary Floating-poin Arithmetic, ANSI/IEEE Standard 754.

Observe, através do programa abaixo, um curioso valor estabelecido pelo padrão IEEE 754. O valor é NaN, abreviação de Not-a-Number, que em português é algo como "não é um número". Este valor é produzido quando se divide zero por zero ou é fornecido pela constante Float.NaN, conforme o programa abaixo.

package ieee754;
public class NaN {
    public static void main(String[] args) {

        float f = 0.0f/0;


        System.out.println(Float.NaN);
        System.out.println(f);
    }
}
