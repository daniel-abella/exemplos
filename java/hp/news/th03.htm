Interface Runnable e classe Thread

Em Java, a execução de uma atividade pode dar início à execução de outra atividade, sem que a primeira seja bloqueada, através da criação de um thread. A criação de um thread exige uma classe que: (a) implementa a interface Runnable ou (b) herda da classe Thread. Não há outra forma de se criar threads em Java. Estas duas opções são ilustradas abaixo: 

class MinhaClasse implements Runnable { 
    /* implementa public void run() */ 
}

ou

class MinhaClasse extends Thread {
    /* sobrepõe public void run() */ 
}
