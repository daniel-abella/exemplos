Soluções de exercícios!

O arquivo codigo.jar contém as respostas para todos os exercícios fornecidos até o presente. Após obter este arquivo, no diretório onde deseja depositar as respostas, execute 

jar xvf codigo.jar
Arquivos .jar são arquivos que reúnem outros arquivos. Trata-se de uma forma de "empacotar" vários arquivos em um único. Por exemplo, as respostas para os exercícios é fornecida através de várias classes, distribuídas em vários arquivos e em diretórios distintos. Através de um jar file, todos eles podem ser reunidos em um único arquivo, o que facilita a distribuição.

Mais informações sobre um jar file podem ser obtidas através da documentação local do SDK. Sugiro que faça o download para quando o acesso a internet não for constante.
