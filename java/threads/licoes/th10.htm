Programa concorrente (implementa Runnable)

O programa concorrente ilustrado abaixo foi extensivamente comentado em Programa concorrente (estende Thread), em conseqüência, seguem apenas os comentários correspondentes para a versão corrente que faz uso da interface java.lang.Runnable, em vez de estender a classe java.lang.Thread.

Implementar a interface Runnable significa implementar o método run(), conforme ilustrado abaixo. O método main() cria uma instância de Thread e fornece como argumento uma instância da classe que define a tarefa concorrente por meio do método run(). O thread é criado com a mensagem start().

O novo thread terá como missão executar o método run() que, neste caso, resulta na execução do método tarefa(1). Isto é executado em thread próprio e distinto daquele que requisitou estas operações. O thread que requisitou estas operações prossegue normalmente, sem aguardar pelo término da execução do thread criado e, neste caso, inicia a execução de tarefa(2). Ou seja, tarefa(1) e tarefa(2) serão executados concorrentemente. 

public class Concorrente implements Runnable {
    
    public void tarefa(int prefixo) {
        for (int i = 0; i < 100; i++) {
            System.out.println(prefixo + " " + i);
        }
    }
    public void run() {
        tarefa(1);
    }

    public static void main(String[] args) {
        Concorrente cc = new Concorrente();
        Thread t = new Thread(cc);
        t.start();
        cc.tarefa(2);
    }
}
