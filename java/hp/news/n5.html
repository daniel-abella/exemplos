Operador condicional

O operador condicional é ternário. O primeiro dos três operandos é uma expressão do tipo boolean. As outras duas expressões serão avaliadas exclusivamente, conforme a avaliação da primeira expressão. Os símbolos ? e : são empregados na sintaxe deste operador conforme o exemplo abaixo. 

public class OperadorCondicional {
    public static void main(String[] args) {
        String s = (args.length != 0) ? "SIM" : "NAO";
        System.out.println("ALGUM ARGUMENTO FOI FORNECIDO: " + s);
    }
}

Nesta classe é declarada uma variável s do tipo String. O valor inicial desta variável depende da execução do operador condicional. Neste caso, se a condição args.length != 0 for verdadeira, então a literal "SIM" será referenciada por s, senão, a literal "NAO" será empregada.

Em geral, o operador condicional permite compactar expressões sem que a legibilidade seja prejudicada, conforme o exemplo abaixo. As versões apresentadas são equivalentes do ponto de vista funcional, embora aquela empregando o operador condicional seja menor. Muitos provavelmente também consideram a versão empregando o operador condicional mais simples e menos propensa a erros.

Versão if Versão ? : (operador condicional) 
public int abs(int v) {
    if (v < 0) {
        return -v;
    } else {
        return v;
    }
}
 
public int abs(int v) {
    return (v < 0) ? -v : v;
}
