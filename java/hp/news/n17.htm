O método toString()

Toda classe em Java é uma extensão da classe Object, inclusive os arrays. Em conseqüência, toda instância em Java pode receber a mensagem toString(), assim como também os arrays, pois o método String toString() é herdado da classe Object. Não interessa a classe, um objeto desta classe pode receber a mensagem toString(). Se este método não foi sobrecarregado (overloaded), então espere por uma String que não contem muito além do nome da classe do objeto. Este é o comportamento definido na classe Object.

toString() é a forma convencional em Java de se ter acesso ao estado de uma instância através de uma String. Por exemplo, se p é uma referência para uma instância da classe Pessoa, então é razoável esperar que p.toString() contenha informações como o nome e a idade da pessoa em questão.
