O que é um thread?

Um thread representa um fluxo de controle em um programa. Toda vez que um programa é executado, um thread é criado. Uma mesmo programa pode dar origem a vários threads. Neste caso, o programa é dito multithreaded. Threads são executados concorrentemente em Java.

Em Java, quando um programa inicia a execução do método main um thread é criado. Este thread pode criar outros. Se nenhum outro thread é criado, então o programa é dito seqüencial. Caso o método main crie outros threads, então todos estes serão executados concorrentemente.

Thread também é empregado para designar uma instância da classe java.lang.Thread.
