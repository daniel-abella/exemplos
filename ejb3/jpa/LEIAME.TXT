CLASSES:
- Conteiner
- Elemento
- ConteinerService
- ConteinerRepository

DESCRIÇÃO:
Conteiner é uma entidade que possui um nome e uma lista de instâncias 
de Elemento. Cada Elemento possui um nome. ConteinerService oferece um 
único serviço: converte para maiúsculas o nome de cada Elemento de uma
lista de um contêiner cujo nome é fornecido.

OBSERVAÇÕES:
- Código fonte é fornecido no diretório 'src'
- Não inclui código de teste, que é fornecido em projeto próprio.
- Nome deste projeto: dominio
- Nome do projeto de teste: dominio-teste

PARA CADA CLASSE A SER PERSISTIDA VERIFIQUE SE POSSUI:
- construtor default (pelo menos com visibilidade de package)
- atributo id (private Long id) para armazenar identidade do objeto
- o campo version (private int version) para optimistic lock
- implementação (overriding) dos métodos equals() e hashCode() (em alguns casos).
