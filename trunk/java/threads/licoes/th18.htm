Identificação de um thread

Um thread pode ser identificado por um nome. Há duas formas de se estabelecer o nome de um thread: (a) no momento em que uma instância de Thread é criada ou (b) através do método setName(). Todo thread possui um nome fornecido automaticamente pela máquina virtual Java. Conforme ilustra o programa abaixo, o nome pode ser recuperado através de Thread.currentThread().getName().

public class NomeThread implements Runnable {

    public void run() {
        System.out.println(Thread.currentThread().getName());
    }

    public static void main(String[] args) {
        Thread t = new Thread(new NomeThread());
        System.out.println(t.getName()); // Thread-0
        t.setName("Primeiro nome");
        System.out.println(t.getName()); // Primeiro nome
        t.setName("Segundo nome");
        t.start();                       // Segundo nome
    }
}
