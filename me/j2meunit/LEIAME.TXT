Testes de unidade para aplicações Java ME

Este projeto compreende quatro classes. Duas 
classes que contêm os métodos de teste, 
exemplo.Teste1 e exemplo.Teste2. Outras duas
executam os testes via console, exemplo.Console,
e a outra como um midlet, exemplo.Midlet.

A classe exemplo.Console é executada como uma
aplicação Java convencional. A classe exemplo.Midlet
deve ser executada como um MIDlet. 

OBSERVAÇÕES PARA AS CLASSES DE TESTE:
1. A classe de teste deve estender 'j2meunit.framework.TestCase'
2. Crie o construtor default
3. Crie o construtor com dois argumentos: String e TestMethod.
   A única operação é chamar super(s,tm).
4. Crie os métodos de teste (contendo assertions).
5. Crie o método 'Test suite()'. Veja exemplos em 
   exemplo.Teste1 e exemplo.Teste2.
   
OBSERVAÇÕES PARA EXECUÇÃO VIA MIDLET:
1. Crie classe que estende 'j2meunit.midletui.TestRunner'
2. Crie o método 'startApp()' conforme ilustrado em 'exemplo.Midlet'

OBSERVAÇÕES PARA EXECUÇÃO VIA CONSOLE: 
1. Crie uma classe que estende 'j2meunit.framework.TestCase'  
2. Crie o método 'Test suit()' conforme ilustrado em 'exemplo.Console'.
3. Crie o método 'main' no qual TestRunner.main(String[]) é
   executado. O argumento contém nome da classe cujo método suite
   será executado.
