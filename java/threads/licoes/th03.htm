Interface Runnable e classe Thread

Em Java, a execução concorrente de atividades é possível por meio da criação de threads. Para que um thread possa ser criado é necessário identificar o que será executado concorrentemente. Isto pode ser feito de duas maneiras: (a) implementação da interface java.lang.Runnable ou (b) extenção da classe java.lang.Thread. Os exemplos abaixo ilustram as opções possíveis.

class MinhaTarefa implements Runnable { 
    public void run() { 
        System.out.println("tarefa concorrente");
    }
}

ou

class MinhaTarefa extends Thread {
    public void run() { 
        System.out.println("tarefa concorrente");
    }
}
