Um programa faz uso de pelo menos um thread

Todo programa em Java dá origem a pelo menos um thread. Convença-se de que o método public static void main(String[] args), quando chamado pela máquina virtual Java, é executado no thread criado para a execução deste programa. Naturalmente, o código executado pode criar outros threads. O programa abaixo envia várias mensagens ao thread correspondente ao método public static void main(String[] args).

public class ThreadMain {
    public static void main(String[] args) {
        Thread t = Thread.currentThread();   // Obtém thread corrente
        System.out.println(t.getName());     // Imprime nome
        System.out.println(t.getPriority()); // prioridade
        System.out.println(t.toString());    // nome
        
        try { // dorme por 1 segundo
            t.sleep(1000);
        } catch (InterruptedException ex) {
        }

        System.out.println(t.activeCount()); // 1
        System.out.println(t.isAlive());     // true

        t.yield(); // cede vez para "outros"
    }
}
