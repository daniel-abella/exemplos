Criação de um thread via classe Thread

Concorrência em Java pode ser implementada através do emprego de threads.

O primeiro passo da criação de um thread é a herança da classe java.lang.Thread ou implementação da interface java.lang.Runnable. Em ambos os casos deve-se sobrepor o método public void run(). É através deste método que a tarefa a ser realizada de forma concorrente, em seu próprio thread, é especificada.

O segundo passo é criar uma instância da classe que herda de Thread e sobrepõe o método public void run(). A criação desta instância não cria um thread. Criar uma instância da classe Thread não cria um thread.

O terceiro e último passo, o ato principal, é enviar a mensagem start() para a instância criada. É a execução do método start() que cria um thread. Se esta mensagem não é enviada, então nenhum thread é criado. Quando a mensagem é enviada, contudo, a festa se inicia: um novo thread é criado e passa a executar o que foi declarado no método run(). Tudo isto de forma concorrente com o thread que estava em execução, ou seja, aquele que criou uma instância da classe derivada e enviou a mensagem start().
