Variável de ambiente PATH

O compilador javac e a máquina virtual Java java precisam de valor adequado na variável PATH para uma operação correta.

A variável PATH, caso não inclua o diretório bin do ambiente de desenvolvimento Java instalado, fará com que a execução de javac e de java tenham a seguinte repercussão: Bad command or file name.

Este erro ocorre porque a variável PATH não inclui o diretório bin do ambiente de desenvolvimento Java onde se encontra estes programas. Se o diretório onde tal ambiente está instalado é c:\j2sdk1.4.2, então a variável PATH deve incluir o diretório c:\j2sdk1.4.2\bin. Atualizar o valor desta variável depende do sistema operacional empregado. Para o Windows(r) o comando é:

set path=c:\j2sdk1.4.2\bin;%PATH%

Detalhes das ferramentas do ambiente de desenvolvimento da Sun(tm) estão disponíveis na documentação que pode ser obtida gratuitamente em http://java.sun.com/docs. Sugestão: você pode fazer uso da documentação e utilizá-la localmente, sem a necessidade de conexão com a internet.
