Tarefa concorrente

Em Java, um programa concorrente, seja através da implementação da interface Runnable ou via herança da classe Thread, estabelece as tarefas a serem executadas concorrentemente via método run(). Estas duas formas são ilustradas abaixo.

public class Concorrente extends Thread {
    public void run() {
        /* tarefa a ser executada concorrentemente */
    }
}

Alternativamente a tarefa concorrente pode ser descrita em uma classe que implementa a interface Runnable.

public class Concorrente implements Runnable {
    public void run() {
        /* tarefa a ser executada concorrentemente */
    }
}
