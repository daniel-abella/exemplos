1:N (bidirecional)

Uma instância de Pessoa está associada a zero ou mais instâncias de Endereco. Uma instância de Endereco, caso exista, deve estar associada a exatamente uma instância de Pessoa. A partir de uma Pessoa podemos recuperar todos os endereços correspondentes. No sentido inverso, a partir de um endereço, pode-se obter diretamente a quem este está associado.

Importante: a classe Pessoa fará uso de java.util.Set para manter as relações com as instâncias de Endereco. Em conseqüência, a classe Endereco terá que implementar os métodos hashCode() e equals().
