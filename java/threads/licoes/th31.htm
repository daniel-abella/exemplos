Não existe variável synchronized

Apenas métodos e blocos de código podem ser declarados synchronized. Uma variável não pode ser declarada synchronized. É um erro de compilação declarar uma variável como synchronized. 

public class VariavelSynchronized {
    synchronized int x = 1; // ERRO DE COMPILAÇÃO!!!!
}
