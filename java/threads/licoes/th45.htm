O que significa synchronized?

Um método ou bloco de código synchronized realiza duas operações especiais relevantes para um programa multithreaded: (1) após computar uma referência para um objeto, mas antes de executar o corpo do bloco sincronizado, o lock do objeto associado deve ser adquirido (retido de tal forma que nenhum outro possa adquiri-lo enquanto não for liberado) e (2) após a execução do bloco de código sincronizado, o lock que havia sido retido é liberado de tal forma que outro thread possa adquiri-lo.

Um método ou parte de um método (bloco de código) pode ser declarado com a palavra reservada synchronized de tal forma que o comportamento descrito acima se aplica quando tal código for executado.
