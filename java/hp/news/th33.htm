Execução simultânea de múltiplos objetos

Um método deve ser declarado synchronized apenas quando uma ou mais instâncias da classe em questão são compartilhadas entre pelo menos dois threads. Senão vejamos.

Se há um único thread, então não existe concorrência. Se cada objeto é utilizado por apenas um thread, então não há concorrência sobre este objeto, ou seja, memória não é compartilhada entre threads e, dessa forma, a palavra reservada synchronized é desnecessária e deve ser evita até por eficiência.
